# TemplateControl — Avalonia HMI/SCADA Controls Library

Библиотека кастомных высокопроизводительных пользовательских элементов управления (UI Controls) для создания промышленных HMI (Human-Machine Interface) и SCADA-систем на базе фреймворка **Avalonia UI**.

**Технологический стек:** C# / .NET 8 / Avalonia 12.0.4

---

## Особенности архитектуры (CODEX)

Данная библиотека строго придерживается внутренних архитектурных стандартов (**Avalonia Controls Codex**), гарантирующих максимальную производительность, переиспользуемость и независимость UI от бизнес-логики:
- **Lookless Pattern**: Исключительное использование `TemplatedControl` (без `UserControl`). Визуальное представление полностью отвязано от логики компонента.
- **Изоляция**: Компоненты ничего не знают о базах данных, MVVM-фреймворках или сетевом слое. Данные принимаются через `StyledProperty`, а действия транслируются через `RoutedEvents`.
- **Стилизация через ControlTheme**: Никаких захардкоженных кистей и магических строк. Поддержка системных тем реализована через `DynamicResource`.

## Состав библиотеки (Controls)

### 1. PipeControl
Контрол для отрисовки сложных систем трубопроводов на панелях мнемосхем.
- **3D-шейдинг**: Визуализация цилиндрического объема с помощью градиентов (`IsFilled`, `ActiveColor`).
- **Маршрутизация**: Интерактивный режим "Design Mode" для визуальной расстановки узлов.
- **Фитинги**: Автоматическая генерация отводов (90° / 45° Elbows) и фланцев в точках изгибов.

### 2. NumericPad
Универсальная виртуальная цифровая клавиатура (Numpad) для устройств с сенсорным экраном (POS-терминалы, промышленные панели).
- Обработка `Coerce` для удержания вводимого значения в аппаратных рамках (`Minimum`/`Maximum`).
- Встроенные PseudoClasses для визуализации ошибок ввода (`:error`) и пустого состояния (`:empty`).

### 3. SetValueControl
Интеллектуальный контроллер ввода уставки (Setpoint). Интегрирует внутри себя `NumericPad` (во всплывающем окне `Popup`) и добавляет уровни логики для безопасности на производстве.
- **Адаптивный шаг (Adaptive Step)**: Изменение шага кнопок ± в зависимости от порядка величины значения.
- **Безопасные зоны**: Границы `SafeMinimum` / `SafeMaximum` предотвращают случайный выход за пределы нормального режима работы.
- **Предотвращение критических изменений**: Ограничитель резкого скачка (свыше 20% от текущего значения) с принудительным оверлеем для подтверждения.
- **ML Подсказки**: Интерфейс для вывода рекомендуемого значения, рассчитанного внешними алгоритмами предиктивной аналитики.

## Запуск демо-приложения (VS Code)

Для удобства добавлены задачи для сборки и запуска в VS Code.

1. Откройте проект в VS Code: `code .`
2. Нажмите `Ctrl+Shift+B` для сборки решения.
3. Используйте задачу `run` (доступна через `Terminal -> Run Task -> run`) для старта демонстрационного приложения, где все три контрола показаны в действии на панелях SCADA.
